Chromatic Trainer

Welcome! This is a simple ear-training application.

Instructions:

At the beginning of each turn a sound will play. It is your job to repeat the pattern back.

On each new turn you will only see and hear one new note, while you still have to input the entire melody from the beginning.

Change Sounds:

Each time you start a new game the sounds will change to keep it fun and also challenging.

If you want to change the sounds yourself, simply click on the game title.

Tips:

Practice! Ears in the air!

View The Code:

This is a long-term programming project. I want to combine my skills and expertise in classical music education with my new career. This is a perfect opportunity and a way for me to deliver an educational tool to the public without my continual personal engagement. If you want to check out how it works you can see it here on my Github profile.
